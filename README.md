# base-auth-jwt-api-nestjs

## Installation

```bash
$ npm install
```

## Required environment variables

```bash
PORT=

MONGODB_URI=

MAIL_HOST=
MAIL_PORT=
MAIL_USER=
MAIL_PASSWORD=
MAIL_FROM=

FRONTEND_HOST=
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod


```
